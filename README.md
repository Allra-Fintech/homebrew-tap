# homebrew-tap

Homebrew tap for [Allra-Fintech](https://github.com/Allra-Fintech) tools.

## Usage

```bash
brew tap Allra-Fintech/tap
brew install Allra-Fintech/tap/mdv
```

> **Note:** There is an unrelated Python tool also named `mdv` in homebrew-core.
> Always use the fully-qualified name `Allra-Fintech/tap/mdv` to install this one.

## Formulae

| Formula | Description |
|---------|-------------|
| [mdv](Formula/mdv.rb) | CLI Markdown viewer: browser rendering with live reload, GFM, Mermaid, and PDF export |
